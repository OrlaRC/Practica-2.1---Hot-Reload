# Climate App

Aplicación Flutter desarrollada para la práctica P2.1.

## Objetivo

Verificar la instalación y configuración de Flutter, ejecutar una aplicación en un dispositivo Android y probar la funcionalidad Hot Reload.

## Tecnologías

- Flutter 3.44.0
- Dart
- Android 10

## Ejecución

```bash
flutter pub get
flutter run
```

## Resultado

La aplicación muestra información climática básica y permite visualizar cambios en tiempo real mediante Hot Reload.
